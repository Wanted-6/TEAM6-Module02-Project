현재 역할 전환

현재 선택: STUDENT / 사용자 ID: 1

역할 전환은 로그인 기능 전 임시 테스트용입니다. 현재 선택한 역할과 사용자 ID를 기준으로 화면 노출과 권한 검증을 확인할 수 있습니다.

게시판 메인

LMS 게시판 기능의 시작 화면입니다. 게시판 종류별로 이동하고, 각 문맥에 맞는 등록 화면으로 진입할 수 있습니다.

게시판 종류

관리자 공지

관리자가 작성하고 전체 사용자가 조회하는 전체 공지 게시판입니다.

코스 공지

강사가 코스 단위로 공지를 작성하고 수강생이 확인하는 게시판입니다.

자유게시판

학생과 강사가 자유롭게 글을 쓰고 소통하는 커뮤니티 게시판입니다.
